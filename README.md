# newsletterer
Agentic AI system for automating monthly newsletters given topics and opinion directions from a human marketing agent

# How to use:
Test out the test_driver function by

1. create a .env file with OPENAI_API_KEY='YOUR_API_KEY'
2. python3 -m venv .venv
3. source .venv/bin/activate
4. pip install -r requirements.txt
5. python _driver.py
